# Xposed_OneLineClock
Always use one line clock on Android 12 lock screen

Requirements:
- Android 12
- Xposed Implementation
- ROM should be close to vanilla Android, heavily modified OEM ROMs may not work

Tested on Pixel Experience with LSPosed.
